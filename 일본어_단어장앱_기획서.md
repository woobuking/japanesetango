# 일본어 이미지 단어장 앱 기획서

## 프로젝트 개요
JLPT N5~N1 단어를 이미지와 함께 학습하는 웹앱.
2단계 학습 방식: 이미지+단어 동시 학습 → 이미지만 보고 단어 맞추기.

---

## 기술 스택
- Frontend: HTML + CSS + Vanilla JS (단일 파일)
- DB: Google Sheets (단어 데이터 저장 및 관리)
- 이미지: Unsplash API 자동 매핑
- 배포: Netlify (정적 배포)

---

## 데이터 구조 (Google Sheets)

| word_id | level | japanese | hiragana | korean | english | image_url | category |
|---------|-------|----------|----------|--------|---------|-----------|----------|
| 001 | N5 | 犬 | いぬ | 개 | dog | (자동매핑) | 동물 |
| 002 | N5 | 水 | みず | 물 | water | (자동매핑) | 자연 |

- 초기 데이터: JLPT N5 단어 약 800개부터 시작
- 데이터 출처: GitHub 공개 JLPT wordlist JSON → Sheets 임포트
- image_url: Unsplash API로 english 키워드 검색 후 자동 매핑

## 화면 구성

### 1. 메인 화면
- 레벨 선택: N5 / N4 / N3 / N2 / N1
- 학습 모드 선택: 학습 모드 / 테스트 모드
- 오늘의 학습 현황 (학습한 단어 수 / 정답률)

### 2. 학습 모드 (이미지 + 단어 동시 표시)
- 카드 형태로 1장씩 표시
- 표시 내용: 이미지 + 일본어(한자) + 히라가나 + 한국어 뜻
- 버튼: 알아요 / 몰라요 → 다음 카드
- 진행바 표시 (예: 12/50)

### 3. 테스트 모드 (이미지만 표시 → 단어 맞추기)
- 이미지만 표시
- 4지선다 보기 (정답 1 + 오답 3 랜덤)
- 정답/오답 즉시 피드백
- 오답 노트 자동 저장

### 4. 오답 노트
- 틀린 단어 목록
- 오답 단어만 다시 학습 가능

## 핵심 로직

### 이미지 자동 매핑
```javascript
// Unsplash API로 영어 키워드 검색
const getImage = async (keyword) => {
  const res = await fetch(
    `https://api.unsplash.com/photos/random?query=${keyword}&client_id=YOUR_KEY`
  );
  return res.json();
}
```

### 학습 진도 저장
- localStorage 사용
- 저장 데이터: 학습 완료 단어 ID, 오답 단어 ID, 레벨별 정답률

### 카드 셔플
- Fisher-Yates 알고리즘으로 매 세션마다 랜덤 순서

## 개발 순서

1. Google Sheets에 N5 단어 데이터 세팅
2. Sheets → 앱 데이터 연동 (JSON API)
3. 학습 모드 카드 UI 구현
4. Unsplash API 이미지 자동 매핑
5. 테스트 모드 4지선다 구현
6. 오답 노트 기능
7. 진도 저장 (localStorage)
8. Netlify 배포

---

## 추가 고려사항

- 추상적 단어(예: 自由, 愛情)는 이미지 매핑 대신 일러스트 이모지로 대체
- 오프라인 대비: 한 번 로드된 이미지는 캐싱
- 모바일 최적화 필수 (스와이프로 카드 넘기기)
- 향후 확장: 사용자 계정 추가 시 Firebase로 DB 이전 가능
